# 📱 About Pakka

Pakka is a premium social commitment platform where friends create commitments together by locking a refundable deposit.

If you keep your promise,

✅ Your full deposit is refunded.

If you don't,

❤️ Your deposit is donated to a verified charity **or**

🍽️ contributes toward the group's dinner.

Nobody wins another person's money.

Pakka isn't about punishment.

It's about accountability.

# 🎯 Why Pakka?

People rarely forget plans—they often lack meaningful accountability.

Pakka creates a commitment system that encourages people to follow through while ensuring that missed commitments still create something positive through a shared dinner or a charitable donation.

---

# 🍽️ Dinner Mode

If **Dinner** is selected:

* Everyone who attends receives a **100% refund**.
* Only the deposits of people who miss remain in escrow.
* Those deposits contribute toward the restaurant bill.
* The remaining bill is paid normally by the group outside the app.
* If dinner isn't redeemed within **48 hours**, the forfeited deposits are automatically donated to the selected charity.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/pakka.git
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npx expo start
```

Run on Android

```bash
npx expo run:android
```

Run on iOS

```bash
npx expo run:ios
```
